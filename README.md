# endustriyel-anomali-sistemi
# Endüstriyel Enerji Verimliliği ve Anomali Tespit Sistemi

Bu proje, üretim hattı verilerini simüle ederek sıcaklık, titreşim, akım ve enerji tüketimi üzerinden anomali tespiti yapar.

## Dosyalar
- `data_generator.py` → örnek üretim verisi oluşturur
- `anomaly_detection.py` → anomalileri tespit eder
- `requirements.txt` → gerekli kütüphaneler

## Kurulum
```bash
pip install -r requirements.txt
